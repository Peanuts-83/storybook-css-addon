# Storybook CssViewer Addon

This addon allows you to expose style related to your storie's component in a separate TAB called "CSS". 

Style file formats allowed: *.css, *.less, *.sass, *.scss, *.styl.

<div className="subheading">Configure</div>

<div className="config-setup">
    <i>In Storybook config file <b>.storybook/preview.js|ts</b></i>
    ```javascript
      
      const preview = {
        parameters: {
          cssViewerConfig: {
            /** prefix added to style file name */
            prefix: string
            /** prefix to ignore for style file name */
            ignorePrefix: string
            /** file format css|less|sass|scss|styl */
            format: 'css' | 'less' | 'sass' | 'scss' | 'styl'
            /** stories to ignore */
            ignore: Array<string>
          }          
        }
      }
    ```
</div>
